## Featured Projects

### 1. FDEWS | Financial Distress Prediction

Processed **60GB+ SEC EDGAR filings** to build a financial distress prediction system that identifies early insolvency signals up to 12 months in advance.

**Key highlights:**

* Processed 4.9M+ intermediate records using Polars and Parquet
* Built financial distress prediction using XGBoost
* Integrated FinBERT sentiment from MD&A sections
* Achieved 38% precision and 87% recall on an 8.4% base crash-rate dataset
* Delivered a 4.5x lift over random screening
* Added SHAP explainability for model interpretation
* Deployed live dashboard on Hugging Face Spaces

**Tech Stack:** Polars, Parquet, FinBERT, XGBoost, SHAP, Streamlit, Hugging Face Spaces

---

### 2. Freelancer Risk Center

Built a full-stack cashflow forecasting platform that projects **30-day financial runway** for freelancers dealing with delayed payments and irregular income cycles.

**Key highlights:**

* Built forecasting engine using XGBoost and Random Forest
* Derived scenario ranges from rolling forecast-error variance
* Created FastAPI backend for prediction workflows
* Used Supabase for data storage and secured query execution
* Built Next.js dashboard with Recharts visualizations
* Maintained sub-200ms query execution across dashboard views
* Deployed live frontend on Vercel

**Tech Stack:** FastAPI, XGBoost, Random Forest, Supabase, PostgreSQL, Next.js, TypeScript, Tailwind CSS, Recharts

---

### 3. Loan Approval Prediction

Built a reliable credit risk modeling pipeline focused on leakage-free validation and minority-class performance.

**Key highlights:**

* Rebuilt validation workflow to remove pre-split resampling noise
* Achieved 0.81 macro F1 and 0.71 rejected-class F1
* Benchmarked 11 machine learning classifiers
* Selected CatBoost for stable average performance
* Used feature importance to identify credit approval drivers
* Top drivers included Credit History, Loan Amount, and Applicant Income

**Tech Stack:** Python, Pandas, Scikit-Learn, CatBoost, XGBoost, imbalanced-learn, Matplotlib, Seaborn

---

### 4. Global Water Vulnerability Dashboard

Created a Tableau dashboard analyzing global water vulnerability, infrastructure gaps, and regional disparities.

**Key highlights:**

* Cleaned and structured public water vulnerability data
* Built dashboard views for regional comparison and risk analysis
* Visualized water access, infrastructure gaps, and environmental stress patterns
* Designed stakeholder-friendly charts and geographic insights

**Tech Stack:** Tableau, Excel, Data Cleaning, Dashboarding
